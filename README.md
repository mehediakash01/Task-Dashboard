
# Task Dashboard

A modern, responsive task management dashboard built with React and Framer Motion.

## Features

- **Authentication**: Secure login system with token-based auth
- **Dashboard**: Overview of projects, team members, and analytics
- **Progress Tracking**: Visual progress rings and semi-circle meters
- **Team Management**: View team members and their assigned tasks
- **Analytics**: Weekly performance charts and project statistics
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Smooth Animations**: Motion effects using Framer Motion

## Tech Stack

- **Frontend**: React 18, React Router
- **Styling**: Tailwind CSS, custom CSS animations
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **API Client**: Fetch with custom hooks

## Project Structure

```
src/
├── components/        # Reusable UI components
├── pages/            # Page components
├── routes/           # Route protection
├── context/          # Auth context
├── services/         # API hooks
└── index.css         # Global styles
```

## Components

- **ProgressRing**: Circular progress indicator
- **Sidebar**: Navigation menu with mobile support
- **StatusBadge**: Task status indicator
- **AnalyticsBar**: Weekly performance bars

## Getting Started

1. Install dependencies
2. Set up environment variables
3. Run development server
4. Login with provided credentials
